## 📝 输出格式

生成的备忘录严格遵循以下格式：

```
【X月】
第1周（M.D-M.D）
X.D（星期）
- [ ]
- [ ]
- [ ]
```

**格式细节:**
- 月份标题: 24px大粗体
- 周标题: 24px大粗体 + 日期范围
- 日期标题: 粗体 + 星期指示器
- 待办项: 空的勾选框(☐)

## 🔧 技术实现

### 核心技术栈
- **Node.js v24+**: ESM模块化语法
- **AppleScript**: macOS Notes应用集成
- **HTML生成**: 标准化格式输出
- **Date API**: JavaScript日期计算

### HTML格式示例
```html
<div><b><span style="font-size: 24px">【4月】</span></b><br></div>
<div><br></div>
<div><b><span style="font-size: 24px">第1周（4.1-4.5）</span></b><br></div>
<div><b>4.1（三）</b><br></div>
<ul>
<li></li>
<li></li>
<li></li>
</ul>
```

### AppleScript流程
1. 删除现有备忘录（如果存在）
2. 在"我的Mac"账户→"Notes"文件夹创建新备忘录
3. 设置标题为`【X月】`
4. 设置内容为生成的HTML
5. 返回状态给用户

## ⚠️ 常见问题

### Q: 备忘录没有显示在Notes中
**A**: 
- 重启Notes应用
- 检查"最近删除"文件夹
- 等待iCloud同步
- 验证账户名称是"我的Mac"

### Q: 备忘录中显示HTML标签
**A**: 
- HTML生成错误 - 检查控制台输出
- 验证HTML生成 (`/tmp/create_month_todo.applescript`)
- 手动运行AppleScript

### Q: 星期指示器错误
**A**: 
- 验证系统日期/时间设置
- 检查时区配置
- 确保2026年日历正确
