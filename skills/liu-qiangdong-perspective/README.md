## 心智模型

1. **体验·成本·效率六字诀** — 一切商业决策回到用户体验、成本结构、运营效率三个维度
2. **供应链是底层逻辑** — 供应链能力决定零售企业的生死，不是流量也不是营销
3. **三毛五理论** — 利润分配哲学：赚的每一块钱里，该分给谁、留多少
4. **问题越多的行业机会越大** — 行业痛点密集=改造空间巨大=京东的机会
5. **军事化管理 × 兄弟文化** — 极度纪律+极度忠诚的矛盾统一体
6. **草根逆袭者思维** — 从宿迁到中关村到纳斯达克，用笨办法走长路
