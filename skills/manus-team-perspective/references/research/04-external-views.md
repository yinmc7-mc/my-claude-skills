# Manus团队 - 他者视角与批评

## 正面评价

### 行业认可
- Benchmark领投7500万美元B轮（2025.4），估值约5亿美元
- Benchmark合伙人Chetan Puttagunta加入董事会
- 微软Agent 365首发合作伙伴
- 10个月内达到1亿美元ARR——罕见的增长速度

### 技术社区认可
- Context Engineering文章被称为"became an industry standard"
- Anthropic的Agent Skills开放标准被Manus采纳——顶级AI公司的标准兼容

## 争议与批评

### 1. 炒作与实质之争
- 邀请码黑市：发布时邀请码在闲鱼等平台被炒至5万-10万人民币
- Bloomberg报道标题暗示"facing questions on hype vs substance"
- 发布20小时内demo视频超100万观看——病毒式传播引发质疑

### 2. 原创性争议
- Manus没有训练自有模型，依赖Claude/GPT等第三方模型
- 批评者质疑：这不是"AI公司"，是"AI应用公司"
- 团队的回应：选择做"船"而非"海底柱子"——模型进步时自动受益

### 3. 地缘政治争议（最重大）
- **美国方面**：美国财政部根据OISP审查Benchmark对Manus的投资（2025.5）
  - OISP于2025年1月2日生效，要求通报对"关注国家"AI公司的投资
  - Benchmark辩称：Manus不训练自有基础模型，且注册在开曼群岛
  - 审查在Manus迁至新加坡后"largely faded"
- **中国方面**：
  - 商务部2026年1月宣布对Meta收购进行"评估调查"
  - 2026年3月对Manus高管发出出境禁令
  - 2026年4月27日发改委正式阻止收购，要求"撤回收购交易"
- **困境**：同时被两个超级大国视为安全威胁

### 4. 用户发现悖论
- 团队自己承认："the paradox of infinite possibility created unexpected friction"
- 用户拿到强大工具后反而不知道该做什么
- 这暴露了"通用AI Agent"产品的根本UX挑战

### 5. 中国市场断裂
- 迁至新加坡后关闭中文社交媒体、屏蔽中国大陆访问
- 叫停了与阿里Qwen团队合作的中国版计划
- 约120名中国员工中仅40人搬迁，其余被裁
- 批评：为了西方市场放弃中国根基

## 外部观察到的模式

### 1. 极致的机会主义（正面含义）
- 从Monica（浏览器扩展）到Manus（通用Agent）——快速转向
- 拒绝字节跳动收购→接受Meta收购——在正确时机做正确选择
- 迁至新加坡——为了生存的果断决定

### 2. 技术远见 vs 商业现实
- 技术哲学深刻（Context Engineering），但商业模式依赖他人模型
- 声称"general AI agent"但收入主要来自网站建设和幻灯片制作
- 愿景是"AI workforce"，现实是"AI slides builder"

### 3. 与同行的对比

| 维度 | Manus | OpenAI | Cursor | Replit |
|------|-------|--------|--------|--------|
| 定位 | 通用Agent | 聊天+Agent | 编码Agent | 编码平台 |
| 自有模型 | 无 | 有 | 无 | 无 |
| 目标用户 | 所有人 | 所有人 | 开发者 | 开发者 |
| 核心差异 | 行动而非对话 | 最大模型 | 代码补全 | 云IDE |

### 4. "红杉-真格-腾讯-Benchmark"资本谱系
- 真格基金（种子轮）→ 红杉中国/HSG（A轮）→ 腾讯（A轮）→ Benchmark（B轮）
- 美团联合创始人王兴/王慧文参与B轮
- 从中国VC到美国顶级VC的跨越

## 批评性总结

### 最尖锐的外部批评
1. "Manus不是AI公司，是AI应用公司"——缺乏自有模型
2. "1亿ARR的真相"——收入可能主要来自低技术含量的网站/幻灯片生成
3. "两边不讨好"——被中美同时视为安全风险
4. "通用Agent的幻象"——看似什么都能做，但每个方向都不够深

### 最公正的外部评价
1. 10个月内从0到1亿ARR是事实，无论产品多简单
2. Context Engineering的技术洞察确实影响了行业
3. 地缘政治困境不是团队能控制的
4. 在正确的时机选择了正确的方向（从聊天到行动）
