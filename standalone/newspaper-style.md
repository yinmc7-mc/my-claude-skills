# 报纸风一页纸排版

将内容浓缩为 A4 单页 HTML，纯黑白报纸风格，可直接 `Cmd+P` 导出 PDF 或截图。

## 设计规范

### 排版原则
- **纯黑白**：无彩色，无灰色背景，无阴影，无渐变
- **宋体**：`'Songti SC', 'SimSun', 'STSong', serif`
- **线框分隔**：用 `border` 和 `border-bottom` 代替一切装饰元素
- **紧凑**：行距 1.5-1.7，字号 11-12px，标题 12-18px
- **A4 尺寸**：宽 794px，高 1123px，内边距 28-36px

### 页面容器
```css
.page {
  width: 794px;
  min-height: 1123px;
  margin: 16px auto;
  background: #fff;
  padding: 28px 36px;
}
```

### 核心组件

| 组件 | 用途 | 样式 |
|------|------|------|
| 标题区 | 文章标题 | 18px 宋体，居中，letter-spacing 6px，底部 2px 黑线 |
| 核心结论框 | 一句话结论 | 1px 黑线边框，内边距 10-14px，12-13px 字号 |
| 双栏区 | 并列内容 | grid 两列，中间 1px 黑线分隔，`border-top/bottom` 上下封口 |
| 表格 | 维度/对比 | 1px 黑线边框，表头黑底白字，11px 字号 |
| 时间线 | 里程碑 | flex 横排，每格竖线分隔 |
| 引用 | 原话 | 左侧 2px 黑线，10.5px 字号，灰色文字 |
| 警告框 | 风险提示 | 1px 黑线边框，加粗标题 |
| 页脚 | 出处说明 | 居中，10px 灰色字，上方 1px 灰线 |

### 字号体系
- 页面标题：18px
- 栏目标题（h2）：12px，底部 1px 黑线
- 正文：11.5px
- 引用：10.5px
- 页脚：10px

### 打印适配
```css
@page { size: A4; margin: 0; }
-webkit-print-color-adjust: exact;
print-color-adjust: exact;
```

## 触发条件

用户说以下任意一个时触发：
- "报纸风"
- "报纸排版"
- "一页纸"
- "报纸风格"
- "报纸样式"

## 工作流程

1. 判断内容是否适合一页纸（如果内容过多，提醒用户可能需要删减或分页）
2. 按金字塔原理组织内容：核心结论 → 支撑论据 → 行动细节
3. 使用上述组件搭建页面
4. 生成后自动用 `open` 命令打开
5. 提示用户 `Cmd+P` 导出 PDF
